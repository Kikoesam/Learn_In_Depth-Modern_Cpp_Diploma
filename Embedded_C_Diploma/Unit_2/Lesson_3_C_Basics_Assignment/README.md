### 📘 Problem N3: [Write C Program to Add Two Integers]

#### 🧪 Code Implementation
```c 
#include <stdio.h>
int N1;
int N2;
int main()
{
    printf("Enter two numbers: ");
    scanf("%d %d", &N1, &N2);
    printf("You entered: %d" , N1 + N2);
    return 0;
}
```

Output --> Enter two numbers: 5
           1
